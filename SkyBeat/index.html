Quienes somos

Un espacio modular diseñado para que reproduzcas tu música local con una experiencia elegante, ligera y muy fácil de usar.

SkyBeat Modular está construido con archivos independientes de interfaz, lógica de audio y gestión de lista. Esto permite mantener el proyecto ordenado y ampliable sin perder la ligereza de una página estática.
Nuestro objetivo es unir una presentación visual suave con controles precisos, para que escuchar música se sienta tan fluido como mover una lista bien organizada en tu propio dispositivo.

Nuestra misión

Crear una página agradable, funcional y clara, donde cada interacción sea rápida, intuitiva y visualmente coherente.

La misión de esta página es ofrecerte una forma ordenada de escuchar música descargada: apuntar a la carpeta musicas/, cargar todas las pistas automáticamente y controlar la reproducción con precisión.

Función de la página

Todo está pensado para que controles tu audio como en un reproductor moderno, con una interfaz clara y sin distracciones innecesarias.

  • Lee automáticamente los archivos de audio dentro de musicas/ al ejecutarse con un servidor local.
  • Permite elegir el minuto exacto de reproducción con la barra de progreso.
  • Muestra el tiempo transcurrido, el tiempo total y el tiempo restante de cada pista.
  • Incluye botones para retroceder, adelantar 10 segundos y saltar entre canciones.
  • Mantiene un diseño claro, celeste y blanco, con continuidad visual en toda la página.
  • Usa módulos independientes para UI, audio, lista y carga de archivos, facilitando su mantenimiento.
